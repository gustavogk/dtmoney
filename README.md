# :computer: How to run

```bash
# Clone Repository
$ git clone https://github.com/gustavogk/dtmoney.git
```

```bash
# Install Dependencies
$ yarn

# Run Aplication
$ yarn start
```
Go to http://localhost:3000/ to see the result.
